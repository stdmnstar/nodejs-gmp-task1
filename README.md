### Clone the repository

```bash
git clone https://github.com/stdmnstar/nodejs-gmp-task1.git
```
---

### Install dependencies:

```bash
npm install
```
----

### Run tasks 1, 2:

```bash
npm run task1
npm run task2
```
### Run task 3 (tasks 1,2 with use babel and ES6 modules ):

```bash
npm run task1:babel
npm run task2:babel
```
